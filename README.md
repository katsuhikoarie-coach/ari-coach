# Ari 3.0 — 統合セルフコーチング

アドラー心理学・IFS・ナラティブセラピーなど14の手法を統合した熟練メタコーチ「在り（Ari）3.0」です。

## セットアップ

```bash
# 1. 依存パッケージのインストール
cd ari-coach
npm install

# 2. 環境変数の設定
cp .env.example .env
# .env を開き、ANTHROPIC_API_KEY にあなたの API キーを設定してください
```

## 起動方法

### CLI モード

```bash
npm run cli
```

### Web モード

```bash
npm run web
# ブラウザで http://localhost:3000 を開いてください
```

### 本番ビルド

```bash
npm run build
npm run start:cli   # または
npm run start:web
```

## フェーズ一覧

| フェーズ | 説明 |
|---------|------|
| Phase 0: オープニング | セッションの入口。今日のテーマを探索する |
| Phase 1: 現実の受容 | 現在地をありのままに受け取る |
| Phase 2: 理想の物語 | 「本当はどうありたいか」を探る |
| Phase 3: 内なる声 | IFS 的視点で「でも」「どうせ」の声を聴く |
| Phase 4: 障害の特定 | 理想と現実の間にある障害を明確にする |
| Phase 5: 物語の書き換え | 新しいナラティブを構築する |
| Phase 6: 行動コミットメント | 最初の一歩を決める |
| 危機介入 | 強い苦悩が表現された場合の安全プロトコル |

## コマンド一覧

| コマンド | 説明 |
|---------|------|
| `/summary` | 現在セッションのサマリーを今すぐ生成・表示 |
| `/history` | 過去セッション一覧と各サマリーを表示 |
| `/patterns` | 複数セッションを通じたパターン分析を表示 |
| `/quit` | セッションを保存して終了 |

## データ

セッションデータは `data/sessions/` に `{日付}_{ID}.json` 形式で保存されます。

## 環境変数

| 変数 | 説明 |
|-----|------|
| `ANTHROPIC_API_KEY` | Anthropic API キー（必須） |
| `PORT` | Web サーバーのポート（デフォルト: 3000） |
