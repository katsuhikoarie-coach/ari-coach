# タスク：セッション終了機能の実装

## やること

app.py に、ユーザーが途中でセッションを終了できる機能を追加する。

---

## 仕様

### 終了トリガー

チャット入力欄に以下のいずれかが入力されたとき、セッションを終了する。

- `終了`
- `終わり`
- `終わる`

（前後にスペースや句読点があっても検知すること）

### 終了時の動作

1. コーチングのやり取りを止める
2. 以下のメッセージを表示する：

```
今日もセッションお疲れさまでした。
またいつでも、近くにあるテーマを持って来てください。
```

3. 「新しいセッションを始める」ボタンを表示する
4. ボタンを押したら、会話履歴をリセットして最初の状態に戻る

---

## 実装の注意点

- 終了判定は、AIへのAPI送信より前に行う（無駄なAPI呼び出しをしない）
- 終了メッセージはAIからの返答ではなく、アプリ側で固定テキストとして表示する
- セッションリセット後は、通常のPhase 0（オープニング）の状態に戻す
- 既存の会話履歴（st.session_state）の扱いは現在のコードに合わせること

---
